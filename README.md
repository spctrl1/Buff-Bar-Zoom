# Buff Bar Zoom

A lightweight utility to mirror and magnify specific regions of your screen. Perfect for tracking small buff icons or cooldowns in games like Bee Swarm Simulator.

[![Download](https://img.shields.io/badge/Download-Stable_Release-blue?style=for-the-badge&logo=windows)](https://github.com/spctrl1/Buff-Bar-Zoom/releases/tag/v1.1)

## Installation

1. Click the **Download** button above.
2. Download the `.exe` file from the latest release.
3. Run the file.

### "Windows protected your PC" Warning
Because this app is a custom open-source tool and not digitally signed by a corporatiion you will get a warning when you try to run it.
1. Click **"More info"**.
2. Click **"Run anyway"**.

## How to Use

### Controls
| Key | Action |
| :--- | :--- |
| **F1** | **Run** (Hide settings, show overlays) |
| **F2** | **Setup** (Show settings, hide overlays) |
| **F3** | **Preview** (Show both settings and overlays) |
| **F4** | **Quit** |

### Setup Guide
1. **Select Monitor**: Choose the screen where the game is running.
2. **Adjust Regions**:
   - Enable a region (e.g., Region 1).
   - Use the **X / Y** sliders to move the red box over your buffs.
   - Use **W / H** to resize the box.
3. **Customize**:
   - **Zoom**: Increase magnification to make icons easier to see.
   - **Separate Windows**: Check this if you want each region in its own movable window.
